# Advanced Admin UI

A comprehensive client-side admin tool suite for Holdfast: Nations at War.

## Features

- **Cavalry Visualization** - Visual indicators for cavalry units
- **Rambo Indicator** - Highlights players going rambo
- **AFK Indicator** - Shows AFK players
- **Minimap** - In-game minimap overlay
- **Admin UI** - Enhanced admin interface

## Requirements

- Holdfast Modding Launcher v1.0.80 or higher
- RC (Remote Console) login to use admin features

## Installation

### Option 1: Using Mod Browser (Recommended)

2. Click "ðŸ“¦ Browse & Download Mods"
3. Find "Advanced Admin UI" and click Install

### Option 2: Manual Installation
1. Download the latest `AdvancedAdminUI.dll` from [Releases](https://github.com/Xarkanoth/HoldfastModdingLauncher/releases)
2. Place it in your launcher's `Mods` folder
3. Enable the mod in the launcher
