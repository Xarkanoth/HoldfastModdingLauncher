## Installation

1. Download the latest ZIP from the Releases page
2. Extract to any folder
3. Run `HoldfastModdingLauncher.exe`

## Features

- Automatic BepInEx installation
- Easy mod management (toggle mods on/off)
- **Mod Browser** - Download and update mods directly from GitHub
- Auto-update checking for launcher and mods

## Available Mods

### Advanced Admin UI
A comprehensive client-side admin tool suite for Holdfast: Nations at War.
- Cavalry Visualization
- Rambo Indicator  
- AFK Indicator
- Minimap
- Admin UI

*Requires RC (Remote Console) login to use admin features.*

## Requirements

- [.NET 6.0 Runtime](https://dotnet.microsoft.com/download/dotnet/6.0)
- Holdfast: Nations At War (Steam)

## For Server Admins

The mods included with this launcher are designed for **server administration purposes**. They help admins monitor and manage servers more effectively.
